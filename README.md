# MIREA AES Visualizer

Интерактивный визуализатор алгоритма AES для лабораторных работ.

## Особенности
- Пошаговая визуализация раундов AES-128.
- Поддержка различных режимов (ECB, CBC, CTR, GCM).
- Встроенный теоретический справочник.

## Сборка
Проект использует CMake и Qt6.
1. Откройте проект в CLion.
2. Настройте CMake профиль (Release).
3. Соберите проект.
